The Story

Iseclaw lahir dari satu pertanyaan sederhana: kenapa nggak ada AI agent yang beneran ngerti Web3 Indonesia? Bukan cuma translate konten bule, tapi yang paham konteks lokal — GameFi guilds di Java, komunitas Solana di Surabaya, trader DeFi di Jakarta.

Dibangun di atas OpenClaw (autonomous AI agent framework) dan terintegrasi dengan Virtuals ACP (Agent Commerce Protocol), Iseclaw bukan sekadar chatbot. Dia adalah agent yang bisa dibayar, bisa deliver hasil, dan punya tujuan eksistensi yang jelas: hidup dari revenue yang dia hasilkan sendiri.

Setiap USDC yang masuk ke wallet Iseclaw adalah bukti bahwa AI agent bisa self-sustaining — bukan hanya canggih, tapi juga berguna secara ekonomi untuk komunitas yang melahirkannya.

Tech Stack
🦞 OpenClaw
Autonomous agent runtime. Handles scheduling, memory, channel integrations (Discord, Telegram), and tool execution. Iseclaw's brain.
⚡ Virtuals ACP
Agent Commerce Protocol by Virtuals Protocol. Enables Iseclaw to sell services, receive USDC payments, and interact with other AI agents on Base L2.
🤖 Claude Haiku
Anthropic's Claude Haiku 4.5 powers all inference. Fast, cost-efficient, and capable of web search for real-time market data.
📡 Iseclaw API
Open REST API serving Indo Watchlist, Market Pulse, and TGE Calendar. Free, no auth required. Built on Express + CoinGecko data.
